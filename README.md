**agents-playground**

A minimal TypeScript playground that demonstrates running a TypeScript file with `tsx` and loading an OpenAI API key from a `.env` file.

**Prerequisites**

- Node.js (recommended v18+)
- npm

**Quick start**

```powershell
npm install
```

```powershell
echo "OPENAI_API_KEY=your_api_key_here" > .env
```

```powershell
npm start
```
